SELECTION OF THE CAR

All the cars available are shown, so the user can see which one is the best for him and click on the button "Select" next to the info of the desired car.
When clicking on it, the application will go to another window showing the complete information of the car. There, the user must introduce how many days he wants to rent the car for, and the total price of the renting is calculated at real time.
From this window, if the user is not satisfied with this car, he always is able to go back and keep checking the cars clicking on the button "previous". If the user wants to select this car, he must press the button "next" to be redirected to a window were the data of the client will be taken and the renting will be done.
